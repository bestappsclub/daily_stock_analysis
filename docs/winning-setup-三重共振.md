# 强势趋势选股 —— 三重共振买卖法（学习文档）

> 一套**可重复、带纪律**的趋势交易框架。选股只是一半，活下来靠止损。
> 没有任何系统能保证赢；这套做法的目的是把**胜率 × 盈亏比**拉到你这边。

## 0. 一句话

> **只在「三重共振」点进场，进场即设移动止损，强于大盘(PWR)才做，见顶/破位无条件离场。**

---

## 1. 核心理念：顺强势趋势

经典趋势交易（欧奈尔 / 米奈尔维尼 / 温斯坦）都围绕三件事：

| 你要看的 | 技术含义 | 为什么 |
|---|---|---|
| **头头高、底底高** | 道氏上升结构（更高的高点 + 更高的低点） | 趋势的定义本身；结构一破就是趋势转弱 |
| **均线多排** | MA5 > MA10 > MA20（多头排列） | 各周期资金一致向上，回调有支撑 |
| **爆量 / 放量突破** | 突破时成交量明显放大 | 量是真突破的「确认」，无量突破多假 |

**单看任何一个都噪音很大**。真正高胜率的是**三者同时成立的共振点**——再叠加「**温斯坦阶段2**（主升段）+ **领涨 RS / 高 PWR**（强于大盘）」就是教科书买点。NDU 的 `$Ultimate Bull` 本质也是这种组合拳。

---

## 2. 三重共振进场（系统里的 `power_setup` 规则）

DSA 选股引擎新增了一条把三条 **AND** 在一起的规则：

```
头头高底底高(structure=bull)  AND  均线多排(多头趋势)  AND  放量上涨
        ↓ 排序
  阶段2 优先 → 领涨RS优先 → PWR高优先 → 评分高
        ↓
   最「教科书」的票排在最前；命中为空 = 今天没有买点（就别硬做）
```

- 前端【选股】页 → 规则下拉选 **「三重共振·强势突破」** → 运行。
- 实现：`src/services/us_screener_service.py` 的 `_STRATEGY_TEMPLATES`（`power_setup`）+ `_apply_strategy`。
- 单测：`tests/test_screener_stage_smi.py::ApplyStrategyTest::test_power_setup_requires_triple_confluence`。

## 3. 完整闭环：选 → 进 → 守 → 出

| 步骤 | 做什么 | 用系统里的 |
|---|---|---|
| **1 选** | 跑「三重共振」，**只挑 PWR 高（强于大盘）** 的前几名 | `power_setup` 规则 + PWR 列 |
| **2 进** | 进场**当下立刻**设止损位（别等套了再想） | 吊灯止损 / DK 12% 移动止损（`chandelier_stop` / `dk_trail_stop`） |
| **3 守** | 涨就让移动止损跟着抬，**不手动乱动**；跌到止损就走 | 移动止损自动抬升 |
| **4 出** | **阶段转 3/4** 或 **「见顶卖出」信号** 出现 → 减仓 / 清仓 | `stage3/4` 徽章、`top_signal` 规则 |

> 进场决定盈亏比，**止损决定你能不能活到下一次盈利**。多数人亏在没有第 2、4 步。

---

## 4. 风控铁律（比选股更重要）

1. **永远带止损**——进场即设，机械执行，不找借口移到更低。
2. **单笔风险固定**——例如每笔最多亏总资金的 1–2%；用「止损距离」反推仓位，而不是拍脑袋买多少股。
3. **只做强于大盘的**——PWR 低 / RS 落后的，再「便宜」也不碰（弱势股反弹是陷阱）。
4. **不逆势抄底**——阶段4（均线下行）的票不做多；要等阶段1→2 转折确认。
5. **空仓也是一种持仓**——「三重共振」命中为空时，市场没机会，拿着现金等。

---

## 5. 指标速查（这套系统产出的关键字段）

| 字段 | 含义 | 怎么用 |
|---|---|---|
| **PWR** | 相对强度百分位 0–100（对照大盘，= NDU PWR） | 只做 PWR 高的；80+ 为强 |
| **rs_1m / rs_3m / rs_6m** | 1/3/6 月相对大盘超额收益% | 6 月高 + 1 月仍升 = 持续领涨 |
| **Stage（S1–S4）** | 温斯坦四阶段（30 周线斜率+价位） | 只在 **S2** 做多；S3/S4 减/避 |
| **SMI** | 资金流（Chaikin Money Flow，聪明钱代理） | >0 净流入；配合放量看真买盘 |
| **structure** | 道氏摆动结构（bull=头头高底底高） | 共振的「结构」腿 |

---

## 6. 心态

- 交易是**概率游戏**：一套有正期望的规则，机械重复，靠大数定律赚钱。
- **亏损是成本**，不是错误——只要你按规则止损，单次亏损就是做生意的成本。
- 真正的错误是：不带止损、逆势加仓、看到信号不执行、赢一点就跑亏死扛。

> 把「三重共振 + 移动止损 + 只做强势」变成不带情绪的流程，比预测明天涨跌重要一万倍。
